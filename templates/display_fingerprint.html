Collecte de votre empreinte

L'objectif est de collecter l'empreinte de votre navigateur lorsque vous utilisez une extension de défense contre le fingerprinting. Vous pouvez tester avec Random Agent Spoofer, Canvas defender, Canvas Blocker, Ultimate User Agent, Blink.

Dans le champ ci-dessous saisissez la contre mesure que vous utilisez (Random agent spoofer, canva defender, etc) et cliquez sur Envoyer pour envoyer votre empreinte. Ne vous souciez pas de l'ortographe exacte du nom de l'extension.

Pour les personnes sous Windows ou Mac, une visite avec Internet Explorer ou Safari, même sans extension serait appréciée.
