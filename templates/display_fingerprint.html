Collecte de votre empreinte

L'objectif est de collecter l'empreinte de votre navigateur lorsque vous utilisez une extension de défense contre le fingerprinting.

Les extensions sont disponibles aux liens suivants:

  • Random Agent spoofer (Firefox)
  • User agent Overrider (Firefox)
  • User agent Switcher (Chrome)
  • Canvas Defender (liens en bas, Chrome et Firefox)
  • Canvas Fingerprint Block (Chrome)
  • Canvas Blocker (Firefox)

Dans le formulaire ci-dessous, sélectionnez la contre mesure que vous utilisez (veuillez n'en utiliser qu'une à la fois au maximum). De plus, n'hésitez pas à visiter plusieurs fois la page avec la même contre mesure lorsque celle-ci permet de changer la configuration (par exemple changer le profil dans Random Agent Spoofer).

Pour les personnes sous Windows, Mac OS X ou iOS, une visite avec Internet Explorer ou Safari, même sans extension serait appréciée.

Attention

Pour certains extensions telles que Random Agent Spoofer, User Agent Overrider, ..., il est nécessaire de cliquer sur l'icone de l'extension et de choisir le user agent que l'on souhaite utiliser.
